# Support

## What this repository is for

`AlphaXiv-Paper-Lookup` is a focused OpenClaw skill repository for quickly looking up papers via alphaXiv and arXiv.

## Where to ask for help

- open a GitHub issue for reproducible bugs or feature requests
- contact the maintainer directly for sensitive problems

## Before opening an issue

Please include:

- input type: paper id / arXiv URL / alphaXiv URL
- exact command or usage path
- expected result
- actual result
- whether alphaXiv fetch failed or fell back to arXiv
